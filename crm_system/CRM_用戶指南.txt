================================
1. 系統概述
================================

CRM 管理系統是一個專為活動管理設計的綜合性管理平台，主要功能包括：

• 活動出席管理
• 會員資料管理
• 教練資料管理
• 財務記錄管理
• 帳戶權限管理

系統特色：
- 直觀易用的網頁介面
- 多角色權限管理
- 即時數據統計
- 響應式設計，支援各種設備

================================
2. 快速開始
================================

2.1 系統訪問
-------------
主要訪問網址：
• http://localhost:8080 (本地訪問)
• http://in_crm_system.com:8080 (自定義域名)
• http://www.in_crm_system.com:8080 (自定義域名)

2.2 瀏覽器要求
-------------
• Chrome 80+ / Firefox 75+ / Safari 13+
• 建議使用最新版本的瀏覽器
• 啟用 JavaScript 和 Cookie

2.3 網路要求
-------------
• 確保能訪問本地網路
• 如果使用自定義域名，需要正確配置 hosts 文件

================================
3. 登入系統
================================

3.1 登入頁面
-------------
訪問系統後，如果未登入會自動跳轉到登入頁面

3.2 測試帳戶
-------------
系統提供以下測試帳戶：

管理員帳戶：
• 用戶名：admin
• 密碼：admin123
• 權限：完整系統管理權限

教練帳戶：
• 用戶名：trainer
• 密碼：trainer123
• 權限：教練相關功能

會員帳戶：
• 用戶名：member
• 密碼：member123
• 權限：個人資料查看

3.3 登入步驟
-------------
1. 在登入頁面輸入用戶名和密碼
2. 點擊「登入」按鈕
3. 系統驗證成功後自動跳轉到主頁
4. 如果驗證失敗，會顯示錯誤訊息

3.4 登出系統
-------------
1. 點擊右上角的用戶選單
2. 選擇「登出」選項
3. 系統會清除登入狀態並返回登入頁面

================================
4. 主要功能模組
================================

4.1 首頁儀表板
-------------
功能：系統概覽和快速導航
• 顯示歡迎訊息
• 提供各功能模組的快速連結
• 顯示系統狀態信息

4.2 出席管理
-------------
功能：管理活動出席記錄

子模組：
a) 活動管理
   - 查看所有活動列表
   - 活動詳細信息
   - 活動狀態管理

b) 出席檢查
   - 按日期查看出席記錄
   - 記錄會員出席情況
   - 出席統計報表

c) 新增出席記錄
   - 手動添加出席記錄
   - 選擇活動和會員
   - 記錄出席時間和備註

d) 按姓名查詢
   - 根據會員姓名查詢出席記錄
   - 查看特定會員的出席歷史
   - 出席記錄統計

4.3 會員管理
-------------
功能：管理會員資料和檔案

主要功能：
• 查看會員列表
• 會員資料編輯
• 會員狀態管理
• 會員統計信息

4.4 教練管理
-------------
功能：管理教練資料和活動

主要功能：
• 教練資料管理
• 教練活動分配
• 教練績效統計
• 教練檔案維護

4.5 財務管理
-------------
功能：管理財務記錄和報表

子模組：
a) 財務記錄
   - 查看所有財務記錄
   - 收入支出統計
   - 記錄分類管理

b) 按姓名查詢
   - 根據會員姓名查詢財務記錄
   - 個人財務歷史
   - 財務記錄篩選

c) 新增財務記錄
   - 記錄收入或支出
   - 選擇記錄類型
   - 輸入金額和備註

d) 財務報表
   - 月度財務統計
   - 收入支出分析
   - 會員財務統計
   - 項目分類統計

4.6 帳戶管理
-------------
功能：管理系統用戶帳戶

主要功能：
• 用戶帳戶創建
• 權限等級設置
• 帳戶狀態管理
• 密碼重置

5. 詳細操作說明
================================

5.1 出席記錄操作
----------------

新增出席記錄：
1. 進入「出席管理」→「新增出席記錄」
2. 選擇活動日期
3. 選擇活動項目
4. 選擇會員姓名
5. 記錄出席時間
6. 添加備註（可選）
7. 點擊「保存」按鈕

查詢出席記錄：
1. 進入「出席管理」→「出席檢查」
2. 選擇查詢日期
3. 系統顯示該日期的所有出席記錄
4. 可以按活動或會員篩選

5.2 財務記錄操作
----------------

新增財務記錄：
1. 進入「財務管理」→「新增財務記錄」
2. 選擇記錄類型（收入/支出）
3. 選擇會員姓名
4. 輸入項目名稱
5. 輸入詳細說明
6. 輸入地點（可選）
7. 輸入單價和數量
8. 系統自動計算總金額
9. 選擇記錄日期
10. 點擊「保存」按鈕

查看財務報表：
1. 進入「財務管理」→「財務報表」
2. 選擇時間範圍（本月/上月/本季度/本年）
3. 系統顯示統計數據
4. 包括總收入、總支出、淨利潤等

5.3 會員資料操作
----------------

查看會員資料：
1. 進入「會員管理」→「會員檔案」
2. 系統顯示會員列表
3. 可以按角色篩選
4. 點擊會員姓名查看詳細資料

編輯會員資料：
1. 在會員詳細頁面點擊「編輯」按鈕
2. 修改相關信息
3. 點擊「保存」按鈕

5.4 教練資料操作
----------------

查看教練資料：
1. 進入「教練管理」→「教練檔案」
2. 系統顯示教練列表
3. 點擊教練姓名查看詳細資料

查看教練活動：
1. 在教練詳細頁面查看分配的活動
2. 查看活動統計信息
3. 查看教練績效數據

6. 常見問題
================================

6.1 登入問題
------------

Q: 無法登入系統怎麼辦？
A: 
1. 檢查用戶名和密碼是否正確
2. 確認帳戶是否被禁用
3. 清除瀏覽器快取和 Cookie
4. 嘗試使用無痕模式
5. 聯繫系統管理員

Q: 忘記密碼怎麼辦？
A: 聯繫系統管理員重置密碼

6.2 功能使用問題
----------------

Q: 無法新增出席記錄？
A: 
1. 確認已選擇活動日期
2. 確認已選擇活動項目
3. 確認已選擇會員
4. 檢查必填欄位是否完整

Q: 財務報表顯示不正確？
A: 
1. 確認選擇的時間範圍
2. 檢查財務記錄的日期設置
3. 確認記錄類型分類正確

6.3 系統訪問問題
----------------

Q: 無法訪問系統？
A: 
1. 檢查網路連接
2. 確認系統服務是否運行
3. 檢查瀏覽器設置
4. 嘗試使用不同的訪問網址

Q: 頁面載入緩慢？
A: 
1. 檢查網路速度
2. 清除瀏覽器快取
3. 關閉不必要的瀏覽器標籤
4. 聯繫系統管理員檢查系統性能
